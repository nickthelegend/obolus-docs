# Obolus Network Documentation

Official technical documentation for the Obolus Network ecosystem.

- **Status:** Active
- **Framework:** Next.js + MDX
- **Website:** [docs.obolus.network](https://docs.obolus.network)

## 🚀 Overview

Obolus Network is a privacy-preserving RWA yield and credit protocol built on the BNB Chain, providing a secure "Dumb Store" architecture and enclave-verified settlement.

### Key Sections:
- **Privacy Architecture:** ECIES encryption and Chainlink CRE/TEE workflows.
- **Protocol Modules:** Smart Vaults, Hybrid AMM, and Confidential Lending.
- **Developer Guide:** API reference and Settler integration.

## 🛠️ Development

### Prerequisites
- Node.js 18+
- npm / pnpm / yarn

### Local Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
